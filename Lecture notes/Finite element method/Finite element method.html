The Finite Element Method#

The Finite Element Method (FEM) is a very popular method with a wide range of applications. It is an integral technique, similar to the Finite Volume Method which provide it with a number of benefits.

  • Complex geometries

  • Irregular meshing

  • Irregular material properties

  • Options to tweak the method according to the physics

  • (quasi)dynamic mesh refinement

  • Highly flexible boundary conditions

  • Ability to solve boundary physics

  • Excellent accuracy

Consider an ODE \( y^\prime = -y \quad y(0) = 1\), which has an analytical answer \(y = e^{-t}\).

We wish to approximate this function numerically between \([0,1]\) with a line defined by the boundary points, \(f(0) = a = 1\) and \(f(1) = b\), $\(f(x) = 1+x [b-1]\)$

How do we find \(b\)?

Option 1: Following the Finite Difference method, we can subsitute \(f(1)\) into the ODE and solve for b:

\[\begin{split}\begin{align} f'(1) &= f(1) \\ b-1 &= -b \\ b &= 1/2 \end{align}\end{split}\]

and the approximate solution is \(f = 1 -\frac{x}{2}\)

import numpy as np
import matplotlib.pyplot as plt

# Define the functions
f = lambda x: 1 - x / 2
y = lambda t: np.exp(-t)

# Generate values
x = np.linspace(0, 1, 100)
f_values = f(x)
y_values = y(x)

# Plot the functions
plt.plot(x, f_values, label='f(x) = 1 - x/2')
plt.plot(x, y_values, label='y = e^{-t}')
plt.xlabel('x')
plt.ylabel('Value')
plt.legend()
plt.title('Plot of f(x) and y(t) from 0 to 1')
plt.show()

Option 2: Let’s integrate the residual of f(x) plugged into the ODE, \(y^\prime + y\)

\[\begin{split}\begin{align} R &= f^\prime(x) +f(x) \\ &= [b-1] + 1+x[b-1] \\ &= b+x[b-1] \\ R^2 &= b^2 + 2 b x[b - 1] + [x[b - 1]]^2 \\ F = \int_0^1 R^2 \ dx &= 1/3 (7 b^2 - 5 b + 1) \\ \end{align} \end{split}\]

Note that in the end the integral of the residual, \(F\) is a function of the parameters. In this case, it is a parabola in \(b\) which has a minimum at \(\frac{5}{14}\) and therefore the approximate solution is \(f(x) = 1-\frac{9}{14}x\).

We see that option 2 does a better job of approximating the function in the domain since it focusses on the integral of the residual, but it involved some analytical calculus which we would rather avoid.

The Method of Weighted Residuals#

You may have noticed that we did something funny here: we integrated \(R^2\) and then took its derivative with respect to \(b\) in order to find the minimum of the integral! Let’s try to be smarter…

To minimize F(b): $\(\begin{align} \frac{\partial F}{\partial b} = 0 &= \frac{\partial }{\partial b} \int_0^1 R^2 \ dx \\ &= \int_0^1 R \frac{\partial R}{\partial b} \ dx \\ &= \int_0^1 R W \ dx \end{align} \)$

where \(W\) is called a weighting / test function. Deferring the explaination for later, lets take \(W = x\) and integrate:

\[\begin{split}\begin{align} \frac{\partial F}{\partial b} =0 &= \int_0^1 R x \ dx \\ &= \int_0^1 \big[b+[b-1]x\big]x \ dx \\ &= \int_0^1 b x+[b-1]x^2 \ dx \\ &= \bigg[ \frac{1}{6} x^2\big[3b+2[b-1]x\big]\bigg]_0^1 \\ &= b-\frac{2}{5} \\ \end{align} \end{split}\]

which is very close to our previous solution \(b = \frac{5}{14}\) but with a simpler integration.

Aside: relevance of the Method of Weighted Residuals#

The equation involved minimizing an integral quantity

\[F(f) = \int_0^1 R^2 \ dx\]

The quantity \(F\) is called a funcitonal of \(f(x)\) since it depends on not just \(f(x)\) for a particular point but \(f\) over the full domain.

The Calculus of Variations tells us that \(F(f)\) is minimized when any small change in the function \(f(x)\) called the variation \(\delta f\) is zero everywhere in the integration domain. This is useful since functional derivatives follow normal derivation laws!

\[ \delta R^2 = 2R \delta R \]
